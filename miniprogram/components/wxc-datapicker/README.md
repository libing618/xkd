# wxc-datepicker

> 小程序组件 - 日期、时间选择器

## API

### date-picker【props 】

| 名称         | 描述                                                                                                           |
| ------------ | -------------------------------------------------------------------------------------------------------------- |
| `date`       | [说明]：日期选择期中默认选择的时间，如未传，默认选中当前时间                                                   |
| `enableTime` | [说明]：是否允许选择时间，由于微信默认 time 的 picker 仅支持到分，这里同样，有兴趣的同学可自行拓展至支持时分秒 |

### date-picker【bind 】

| 名称           | 描述                                                                                         |
| -------------- | -------------------------------------------------------------------------------------------- |
| `onselectDate` | [说明]：选择时间后点击确定会触发此事件，在使用此组件的 Page，或者 Component 中响应此事件即可 |
